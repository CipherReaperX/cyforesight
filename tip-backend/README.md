# 🛡️ CyForesight - Backend

Enterprise-grade threat intelligence platform backend built with Node.js, TypeScript, Express, PostgreSQL, Redis, and BullMQ.

## 🚀 Features

- **Complete REST API** with 78+ endpoints
- **Authentication & Authorization** (JWT + RBAC)
- **IOC Management** with ML-powered scoring
- **Asset Inventory** with risk assessment
- **CVE Tracking** with vulnerability management
- **MITRE ATT&CK** framework integration
- **Threat Feeds** automated ingestion
- **Real-time Dashboard** with caching
- **Background Workers** for async processing
- **Enrichment Services** (VirusTotal, AbuseIPDB, WHOIS, GeoIP)
- **Recon Tools** integration
- **Comprehensive Logging** with Winston

## 📋 Prerequisites

- Node.js 18+ and npm
- PostgreSQL 14+
- Redis 7+
- (Optional) External API keys for enrichment
